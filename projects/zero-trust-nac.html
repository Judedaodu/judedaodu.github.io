Security Engineering · Project 01

Enterprise Zero-Trust Network Authentication Platform

Network Access Control with FreeRADIUS, Headwind MDM, & 802.1X Architecture

Project Overview

Designed and engineered an enterprise-grade Zero-Trust Network Access Control (NAC) architecture to secure network entry points across a distributed corporate environment serving over 5,000 users. The project mitigated severe risks associated with unauthorized rogue devices, credential sharing, and unmanaged endpoints by implementing strict 802.1X authentication protocols. By integrating a centralized FreeRADIUS engine running on hardened Linux servers with Headwind Mobile Device Management (MDM), the platform automated secure device onboarding, enforced strict least-privilege dynamic VLAN assignments, and established continuous endpoint posture validation. This comprehensive implementation successfully eliminated perimeter vulnerabilities, streamlined compliance auditing, and achieved an estimated $1 million in cost savings relative to commercial proprietary NAC licensing and mitigation overhead.

Problems & Goals

Problems

  • ✗ Unmanaged Perimeter Risks: The absence of network access control allowed unvetted personal devices and third-party hardware to connect directly to the corporate network, increasing exposure to lateral malware movement and data exfiltration.
  • ✗ Credential Sharing & Weak Auth: Reliance on static pre-shared keys (PSKs) or weak legacy authentication protocols led to widespread credential sharing among personnel, leaving network access logs unauditable.
  • ✗ Manual Onboarding Bottlenecks: Provisioning network certificates and security configurations manually for thousands of corporate mobile devices and workstations placed an unsustainable operational burden on the IT infrastructure team.
  • ✗ Lack of Compliance Enforcement: Endpoints frequently accessed the corporate network without verified security postures (e.g., missing device encryption, outdated OS patches, disabled firewalls), violating enterprise security policies.

Goals

  • ✓ Deploy Robust 802.1X Authentication: Implement a vendor-agnostic, open-source RADIUS infrastructure to authenticate every wired and wireless connection attempt at the network edge.
  • ✓ Automate Device Lifecycle Management: Seamlessly integrate MDM orchestration to provision network profiles, cryptographic keys, and configuration updates over-the-air (OTA) to 5,000+ endpoints.
  • ✓ Enforce Zero-Trust Network Isolation: Implement dynamic VLAN steering based on user roles and device ownership to restrict network access strictly to authorized resources.
  • ✓ Optimize Total Cost of Ownership (TCO): Build the core security infrastructure utilizing robust open-source systems to eliminate costly proprietary software licensing fees while exceeding enterprise scaling needs.

Role & Responsibilities

  • ▸ Architected Core NAC Topology: Formulated the end-to-end authentication flow, encompassing the directory services integration, EAP-TLS/PEAP protocol configurations, and RADIUS accounting pipelines.
  • ▸ Deployed and Hardened FreeRADIUS Engine: Built, configured, and optimized FreeRADIUS instances on high-availability Linux servers, tailoring custom authentication modules and dictionary extensions.
  • ▸ Managed MDM Infrastructure: Directed the deployment of Headwind MDM, designing unified device management templates, application blacklisting rules, and automated configuration enrollment workflows.
  • ▸ Configured Network Infrastructure: Partnered with network teams to configure enterprise switches and wireless LAN controllers (WLCs) for 802.1X enforcement, RADIUS fallback behaviors, and dynamic access control lists (dACLs).
  • ▸ Automated Posture Assessment: Wrote validation hooks and scripts to verify that connecting devices met compliance baselines before granting full corporate intranet access.

Architecture & Deployment Flow

  • → Device Enrollment: Corporate endpoints are enrolled into Headwind MDM, which automatically pushes pre-configured enterprise Wi-Fi and wired profiles along with secure connection profiles.
  • → Access Attempt Trigger: When an endpoint connects to an enterprise switchport or SSID, the network access device (switch/WLC) blocks standard traffic and initiates an 802.1X authentication challenge.
  • → EAP Request Forwarding: The network switch packages the client's credentials or digital certificate inside an Extensible Authentication Protocol (EAP) frame and forwards it as a RADIUS Access-Request to the FreeRADIUS server.
  • → Directory Verification: The FreeRADIUS engine processes the request, validating the certificate chain or checking the credentials against the central enterprise directory (LDAP/Active Directory).
  • → Posture and Compliance Validation: Simultaneously, FreeRADIUS queries the Headwind MDM database to confirm the device is active, managed, and compliant with corporate security baselines.
  • → Dynamic Access Grant: Upon successful authentication and posture verification, FreeRADIUS responds with a RADIUS Access-Accept, attaching specific vendor-specific attributes (VSAs) indicating the user's assigned VLAN and ACL.
  • → Network Segment Enforcement: The network switch moves the port or wireless session into the designated secure VLAN, confining the user strictly to their authorized network segment.
  • → Continuous Monitoring & Accounting: The RADIUS accounting loop tracks session durations, byte counts, and disconnections, logging comprehensive telemetry data to a central syslog engine for audit trails.

Key Implementation Details

High-Performance FreeRADIUS Engineering

Customized internal RADIUS logic to handle thousands of concurrent authentication requests with sub-second latency. Implemented robust session caching, structured connection pooling, and multi-tier failover groups to ensure uninterrupted network access across all corporate sites.

Automated MDM-Driven Device Onboarding

Leveraging Headwind MDM, created silent installation payloads that automatically configured network adapters, enforced zero-touch client configurations, and eliminated the need for end-users to interact with complex network settings.

Dynamic Network Segmentation (VLAN Steering)

Replaced flat network topologies with a dynamic identity-driven model. The RADIUS engine dynamically injected VLAN assignments at the hardware layer during authentication, ensuring a guest, an intern, and an executive received separate network access profiles on the exact same infrastructure.

Results & Impact

  • ✓ 100% Zero-Trust Edge Security: Successfully restricted network entry to verified, managed devices only, eliminating unauthorized hardware connections across the entire enterprise environment.
  • ✓ $1 Million in Capital Savings: By architecting and scaling production-grade open-source solutions (FreeRADIUS and Headwind MDM), bypassed exorbitant per-user licensing fees tied to proprietary alternatives.
  • ✓ Seamless Scalability for 5,000+ Users: Built a resilient infrastructure that easily manages the peak load authentication spikes of thousands of corporate endpoints with zero service degradation.
  • ✓ Zero-Touch IT Overhead Reduction: Automated the entirety of network configuration provisioning via MDM, reducing individual device network troubleshooting tickets to near zero.
